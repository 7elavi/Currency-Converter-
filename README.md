
# Currency Converter (Конвертер валют)

Автор: [Максимов Илья]

## Описание
Приложение для конвертации валют с использованием актуальных курсов через API. Поддерживает сохранение истории запросов в JSON.

## Функционал
- Выбор валют из выпадающего списка.
- Валидация ввода (только положительные числа).
- Таблица истории последних операций.
- Работа с внешним API ExchangeRate-API.

## Инструкция по запуску
1. Зарегистрируйтесь на [exchangerate-api.com](https://www.exchangerate-api.com/) и получите бесплатный API-ключ.
2. Вставьте ваш ключ в переменную `API_KEY` в файле `main.py`.
3. Установите зависимости: `pip install requests`.
4. Запустите программу: `python main.py`.
